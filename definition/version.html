LHEO/XML 2.3

LHEO/XML version 2.3 est une application XML permettant de décrire des offres de formation. LHÉO signifie Langage Harmonisé pour l'Échange d'informations sur l'Offre de formation.

L'espace de noms de LHEO/XML 2.3 est http://lheo.gouv.fr/2.3

Changements

Les changements dans le langage sont décrits dans ce document.

Documentation

  • lheo.pdf, le manuel de référence du langage

Fichiers composant l'application XML

  • lheo.dtd, la DTD du langage
  • lheo.xsd, le Schéma XML du langage
  • lheo-glossaire.html, le glossaire du langage
  • lheo-tables.xml, les tables du langage LHÉO

Autres tables

Les tables présentées ci-dessous ne font pas partie de la norme LHÉO et appartiennent à d'autres référentiels. Elles sont présentées ici à titre d'information car elles peuvent être utilisées dans une représentation d'offre au format LHÉO.

  • dict-code-modalite-pedagogique-V10.xml: la table des codes de modalités pédagogiques issue du FORMACODE V10. Un code de cette table est utilisable dans l'élément de LHÉO: code-modalite-pedagogique, en indiquant dans l'attribut ref la version du FORMACODE V10.
  • dict-public-vise-formacode-V10.xml: la table des publics issue du FORMACODE V10. Un code de cette table est utilisable dans l'élément de LHÉO: code-public-vise, en indiquant dans l'attribut ref la version du FORMACODE V10.
  • dict-NSF-V1.xml: la table NSF.
